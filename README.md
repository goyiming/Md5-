# 汉字转拼音加密工具

## 简介

这是一个简单的汉字转拼音加密工具，使用Python的Tkinter库创建了一个GUI应用。用户可以输入汉字，将其转换为拼音，并进行加密操作。

## 功能

- 输入汉字，并将其转换为对应的拼音。
- 对拼音进行加密操作。
- 支持复制、粘贴、剪切和清空功能。
- 统计输入框中的行数。

## 使用方法

1. 输入要转换的汉字。
2. 点击“转换”按钮将汉字转换为拼音。
3. 可选择执行加密操作，点击“加密”按钮。
4. 使用右键菜单进行复制、粘贴、剪切和清空操作。
5. 查看输入框中的行数统计。
